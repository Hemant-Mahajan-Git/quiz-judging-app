# Quiz Judging App

A full-stack application for managing and judging quiz competitions built with React and Node.js.

## Project Structure

```
quiz-judging-app/
├── backend/              # Node.js/Express backend
├── frontend/             # React frontend
├── .gitignore
├── .env.example
└── README.md
```

## Getting Started

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

## Technologies Used

- **Frontend**: React, Axios, React Router
- **Backend**: Node.js, Express, MongoDB (optional)
- **Tools**: ESLint, Prettier

## Development

Ensure both frontend and backend are running for full functionality.
